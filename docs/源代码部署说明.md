# 源代码部署说明

> 前提：确保您已获取到所有配置，详见：[【DouYinSparkFlow 配置生成器】使用说明](配置生成器使用.md)

源码部署方式较为灵活，可部署在本地设备、云服务器，或青龙/白虎等任务面板环境中。需要注意的是，本项目通过无头浏览器模拟人工操作，运行时会对设备性能和浏览器运行环境有一定要求，具体表现请以实际测试结果为准。

本项目基于 Python 开发。选择源码部署方式，默认您已具备基础的 Python 项目运行与环境配置能力。

## 1. 克隆项目

首先拉取当前仓库到对应设备`https://github.com/2061360308/DouYinSparkFlow.git`

## 2. 安装项目依赖

安装依赖`pip install -r requirements.txt`

## 3. 填写环境变量

首先在项目根目录下创建`.env`文件

之后打开已经填写好的配置生成器页面，点击配置生成器左侧最下方` 复制 .env 配置文件 `按钮，得到内容粘贴到`.env`文件中。

## 4. 运行

项目根目录下执行`python main.py`即可。
